# results/

Curated experiment results worth keeping. Unlike `outputs/` (which captures every run), this is for final or notable results you want to reference later.

## Structure

```
results/
├── figures/    # plots, visualizations, architecture diagrams
├── metrics/    # exported metric summaries, comparison tables
└── reports/    # write-ups, analysis notebooks, presentation materials
```

This directory is gitignored by default. If you want to commit specific results (e.g., final figures for a paper), explicitly `git add` them.
