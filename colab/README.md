# colab/

Google Colab notebooks for quick onboarding without local setup.

## Files

- `setup.ipynb` — Installs dependencies, clones the repo, runs a sample training, and shows outputs. Open directly in Colab.

## Notes

- GPU runtime recommended (Runtime > Change runtime type > GPU)
- Data downloads automatically on first run
- Outputs are saved to the Colab filesystem (mount Drive to persist them)
